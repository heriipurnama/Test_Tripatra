# Procurement App

## Overview
A web application for managing purchase orders with email notifications and reporting features.

## Features
- User Authentication
- Purchase Order Creation and Management
- Email Notifications
- Reporting (Web and PDF)

## Technology Stack
- **Frontend**: ReactJS
- **Backend**: Golang with GraphQL
- **Database**: MongoDB
